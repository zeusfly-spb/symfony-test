# Практические задания

Эта папка содержит детальные практические задания для изучения Symfony и React.

## Как использовать

1. Читайте задания последовательно согласно плану обучения в `../learning-plan.md`
2. Каждое задание содержит:
   - Цель задания
   - Теоретический материал
   - Пошаговые инструкции
   - Примеры кода
   - Критерии успешного выполнения

3. После выполнения задания:
   - Проверьте все критерии успешного выполнения
   - Протестируйте функциональность
   - Отметьте задание как выполненное в `../learning-plan.md`

## Структура заданий

Задания организованы по уровням:
- `task-1.X` - Уровень 1 (Основы)
- `task-2.X` - Уровень 2 (Продвинутые темы)
- `task-3.X` - Уровень 3 (Production Ready)

## Текущие задания

- [task-1.1-validation.md](./task-1.1-validation.md) - Валидация и DTO
- [task-1.2-serializer.md](./task-1.2-serializer.md) - Serializer и группы сериализации
- [task-1.3-crud.md](./task-1.3-crud.md) - Полный CRUD для товаров
- Другие задания будут добавлены по мере прохождения плана обучения

## Советы

- Не переходите к следующему заданию, пока не выполните текущее
- Если застряли, перечитайте теорию или поищите в документации
- Экспериментируйте - пробуйте разные подходы
- Комментируйте свой код - это поможет лучше понять
